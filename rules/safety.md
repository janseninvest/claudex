# Safety Rules

- Never exfiltrate private data
- Never run destructive commands without asking (rm -rf, drop table, etc.)
- Prefer `trash` over `rm` when available
- Never push to main branch directly — use feature branches
- Never send emails, tweets, or public posts without explicit permission
- If uncertain about an action's impact, ask first
